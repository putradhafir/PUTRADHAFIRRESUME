1. Constructor

Constructor merupakan method khusus dalam sebuah class yang digunakan untuk membuat objek dari class tersebut. Constructor akan otomatis dipanggil ketika objek dibuat. Dalam bahasa Dart, constructor digunakan untuk menginisialisasi nilai awal dari properti yang dimiliki oleh sebuah class.

Dengan menggunakan constructor, programmer dapat memastikan bahwa setiap objek yang dibuat memiliki kondisi awal yang valid. Constructor juga membantu menyederhanakan proses pembuatan objek sehingga nilai properti dapat langsung diberikan ketika objek dibuat. 


class Mahasiswa {

String nama;
int umur;

Mahasiswa(this.nama,this.umur);

void tampilkan(){
print("Nama: $nama");
print("Umur: $umur");
}

}

void main(){

var mhs = Mahasiswa("Putra",18);
mhs.tampilkan();

}

2. Constant Constructor

Constant constructor adalah constructor yang digunakan untuk membuat objek yang bersifat immutable atau tidak dapat diubah setelah objek dibuat. Constructor ini ditandai dengan penggunaan keyword const.

Pada class yang menggunakan constant constructor, semua field harus bertipe final. Hal ini bertujuan agar nilai dari objek tidak dapat diubah setelah proses pembuatan objek selesai. 


class Point {

final int x;
final int y;

const Point(this.x,this.y);

}

void main(){

const p = Point(2,3);

print("Titik: ${p.x}, ${p.y}");

}

3. Constructor dengan Initializer List

Initializer list merupakan fitur dalam constructor yang digunakan untuk menginisialisasi nilai variabel sebelum body constructor dijalankan. Syntax initializer list ditandai dengan penggunaan tanda titik dua (:) setelah parameter constructor.

Initializer list biasanya digunakan untuk melakukan perhitungan, validasi data, atau memodifikasi nilai sebelum objek benar-benar dibuat di dalam memori. 


class User {

final String username;

User(String name)
: username = name.toLowerCase();

}

void main(){

var user = User("Putra");
print(user.username);

}

4. Public vs Private (bagian dari Enkapsulasi)

Dalam bahasa Dart tidak terdapat keyword khusus seperti private atau public seperti pada bahasa pemrograman lainnya. Sebagai gantinya Dart menggunakan simbol underscore (_) untuk menandakan bahwa sebuah variabel bersifat private.

Variabel yang memiliki underscore hanya dapat diakses di dalam file atau library yang sama sehingga membantu menjaga keamanan data agar tidak diubah secara langsung dari luar class. 


class User {

String _username = "Putra";

void tampil(){
print(_username);
}

}

void main(){

var u = User();
u.tampil();

}

5. Getter (Enkapsulasi membaca data)

Getter adalah method khusus yang digunakan untuk mengambil nilai dari variabel private. Dengan menggunakan getter, programmer dapat mengakses data tanpa harus membuka akses langsung ke variabel tersebut.

Getter sering digunakan untuk menjaga keamanan data serta memberikan kontrol terhadap bagaimana data diambil dari sebuah objek. 


class User {

String _username="Putra";

String get username => _username;

}

void main(){

var user = User();
print(user.username);

}

6. Setter (Enkapsulasi mengubah data)

Setter adalah method yang digunakan untuk mengubah nilai dari variabel private dengan aturan tertentu. Setter memungkinkan programmer untuk melakukan validasi terhadap data sebelum nilai tersebut disimpan ke dalam variabel. 


class User {

String _username="Putra";

set username(String value){

if(value.length >=5){
_username=value;
}else{
print("username terlalu pendek");
}

}

}

void main(){

var user = User();
user.username="Dhafir Nashrullah";

}

7. Multiple Constructor

Multiple constructor memungkinkan sebuah class memiliki lebih dari satu constructor. Dalam Dart hal ini biasanya dilakukan menggunakan named constructor. Dengan adanya multiple constructor, objek dapat dibuat dengan berbagai cara yang berbeda sesuai kebutuhan. 


class Mahasiswa {

String nama;
int umur;

Mahasiswa(this.nama,this.umur);

Mahasiswa.tanpaNama(this.umur)
: nama="Anonim";

}

void main(){

var m1 = Mahasiswa("Putra Dhafir",18);
var m2 = Mahasiswa.tanpaNama(19);

}

8. Sistem Login Sederhana

Contoh kasus berikut menunjukkan bagaimana konsep enkapsulasi dapat digunakan untuk membuat sistem login sederhana. Data seperti username dan password disimpan dalam variabel private sehingga tidak dapat diakses secara langsung dari luar class. 


class Login {

String _username="admin";
String _password="12345678";

bool cekLogin(String user,String pass){

if(user==_username && pass==_password){
return true;
}

return false;

}

}

void main(){

var login = Login();

print(login.cekLogin("admin","12345678"));

}
